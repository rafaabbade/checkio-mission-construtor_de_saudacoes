Como juntar strings em Python?

Você pode usar o operador + para somar strings. Por exemplo: 

nome = Rafa
cumprimento = "Bom dia, " + nome + "! Espero que tenha uma ótima aula."

Existe outra forma de fazer isso?

Sim, você pode usar f-strings para formatar strings de uma maneira mais simples e eficiente. Coloque um f antes da string e use chaves {} para incluir variáveis dentro da string. Por exemplo: 

nome = Rafa
mensagem = f"Olá, {nome}! Seu pedido de {produto} será entregue em breve."
As f-strings são uma forma concisa e legível de juntar variáveis dentro de strings.
